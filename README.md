# John's Recipes

A simple static recipe website designed to be served with GitHub Pages.

## Live site

Once GitHub Pages is enabled, the site will be available at:

https://minecraftman04.github.io/Recipes/

## Current recipes

- [King Edward Deep-Fried Chips](recipes/king-edward-deep-fried-chips.html)

## How to add another recipe

1. Create a new HTML file inside the `recipes` folder.
2. Copy the structure from `recipes/king-edward-deep-fried-chips.html`.
3. Update the title, ingredients, method, and tips.
4. Add a new recipe card to `index.html` linking to the new page.

## GitHub Pages setup

If Pages is not already enabled:

1. Open the repository on GitHub.
2. Go to **Settings** → **Pages**.
3. Set **Source** to **Deploy from a branch**.
4. Choose branch **main** and folder **/** root.
5. Save.
